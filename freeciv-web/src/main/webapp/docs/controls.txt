<h2>Keyboard Shortcuts for Units</h2>
  a: (a)uto-settler  (settler/worker units)
  A: (A)uto-attack - attack when moving, without asking
  b: (b)uild city  (settler units)
  c: (c)enter map on active unit
  d: (d)o - select unit execution
  E: build airbas(E)
  f: (f)ortify unit  (military units)
  F: build (F)ortress  (settler/worker units)
  g: (g)o to tile  (then left-click mouse to select target tile)
  G: delayed (G)oto (at turn change, reqs: unit with less than full moves)
  h: set unit's (h)omecity  (to city on current tile)
  H: make Hideout
  i: build (i)rrigation or convert terrain  (settler/worker units)
  I: v(I)gil: (I)ntercept and auto-attack adjacent air units
  j: no orders - skip orders for this unit this turn 
  J: cancel orders - clear current orders for this unit
  k: next unit of same (k)ind awaiting orders
  l: (l)oad unit as cargo onto transporting unit
  m: build (m)ine or convert terrain  (settler/worker units)
  n: clean (n)uclear fallout
  N: explode (N)uclear
  o: transf(o)rm terrain  (engineer unit)
  O: c(O)nvert unit
  p: clean (p)ollution  (settler/worker units)
  p: (p)aradrop (paratroopers only)
  P: (P)illage  (destroy terrain alteration)
  q: build (q)uay
  Q: select all units from the (Q)ueue awaiting orders (also filters multi-selections)
  r: build (r)oad/railroad  (settler/worker units)
  s: (s)entry unit
  S: un(S)entry all units on tile
  L: unit go (t)o/airlift to city
  t: unload transport (in City, Naval Base, or Quay)
  u: (u)nload unit from transporter
  U: (u)nload, show, and activate all cargo units on transport 
  v: select all units on same tile 
  V: select all units of same TYPE on tile 
alt-v: select all units NOT of this type on tile
  x: unit auto e(x)plore
  X: select all units of same TYPE globally
  w: (w)ait: focus on next unit
  w: <in other tab or pop-up dialog>: close dialog
ESC: <In MAP tab>: Exit GOTO mode/Drag mode/Multi-select mode
ESC: <In other tab>: Exit TAB, return to map.

<h2>Unit Movement:</h2> Numpad, Cursors:
  1: move south-west (alt-.)
  2: move south (alt-,)
  3: move south-east (alt-m)
  4: move west (alt-j)
  5: return focus to last focused unit (shift-<)
  6: move east (alt-l)
  7: move north-west (alt-u)
  8: move north (alt-i)
  9: move north-east (alt-o)
double-tap drag: move unit 

<h2>Other Keyboard Shortcuts:</h2>
b: buy production in city under mouse cursor
w: return to map: close tab/window/pop-up
space: de-select units, exit GOTO mode
shift-space: re-focus to previous map position
ctrl-alt-space: paste smartlink to tile/unit under cursor
ctrl-B: color-fill national territores
shift-C: select all same type units on (c)ontinent
ctrl-C:  city labels on/off
ctrl-alt-D: disconnect (disconnected viewing mode)
ctrl-G: map grid on/off
ctrl-L:  show airlift counter on cities
shift-M: show unit moves left percentage on map
shift-R: revolution: choose new government 
alt-S: fullscreen mode
ctrl-S: save game
shift-T: change tax rates
shift-W: show city tile production on map
alt-W: war calculator
shift-Return: Turn done
&apos; &nbsp; Focus on chat input area. (Restores if minimized)
> &nbsp; Focus/find next dormant unit

<h2>Main Map (Mouse):</h2>
  Left-click on city:            Pop up city dialog
  Left-click on unit:            Select a single unit
Shift-click unit:                Add unit(s) on tile or panel to selection
Ctrl-Shift click tile:           Select tile to be worked by city, on main map.

  Right-click:                   Center tile in view
  Right-click unit:              Unit Orders Menu (pop-up)
  Right-click-and-drag:          Select units by area
  Alt-click-and-drag:            Select units by area (same as above)
Shift-Right-click:               Copy city's current production (or unit on map) to clipboard.
Ctrl-Shift-Right-click:          Paste clipboard into city's current production target. 
Ctrl-Alt-Right-click:            Paste into chat a smartlink to tile or unit under cursor

  Middle-click:                  Show tile info  
  Alt-right click                Show tile info (if mouse has no middle click)
  Double-tap tile:               Show tile info (mobile/touch device only)

  Double-tap map-and-drag:       Move the map 

<h2>Tabs:</h2>
W: map tab; escape window/tab/popup without exiting full screen
alt-C: cities tab
alt-E: empire tab
alt-G: gov. tab
alt-H: help tab
alt-M: map tab 
alt-N: nations tab 
alt-P: prefs tab 
alt-T: tech tab 

<h2>Cities List:</h2>
hover over city name: show all buildings in Super Panel
click Building in Super Panel: change production to Building in hovered city 
right-click Building in Super Panel: buy/sell Building in hovered city; (adds to worklist if can't buy) 
shift-click Building in Super Panel: highlight ON/OFF all cities with this Building 
ctrl-click Building in Super Panel: select/deselect all cities with this Building
space: clear all highlighted and all selected cities

<h2>City Dialog:</h2>
Shift-click specialist:          Change ALL specialists of this type 
Ctrl-click specialist:           Cycle through primary 3 specialists (not secondary specialists)
Click on current production:     Remove current production; move worklist up.                          
  Double-click:                  Add production selection to city worklist / queue (or SHIFT-click)
  Alt-click:                     Change current production to production selection
  Alt-Shift-click                Insert current selection at top of production worklist   
Ctrl-double-click:               Change current city production (selection must already be highlighted)


 SPACE:                          Flip between Main View and Production View
 up arrow or M or V:             Main View
 down arrow or Q:                Production View (Queue)
 O:                              city Options
 left arrow:                     Previous View
 right arrow:                    Next View
 P:                              Preview City
 N:                              Next City
